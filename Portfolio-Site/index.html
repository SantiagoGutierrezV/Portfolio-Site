Personal Projects

Here are some neat projects I have created. Whether it be debugging, languages, frameworks, or even referencing documentation, I have learned something in the process of creating each of these projects.

Modified Snake Game

Redisgned the classic snake game by implementing circular motion and jump functionality. Object-oriented programming principles are used in order to initalize each segment of the snake. Coincidentally, this project was made using Python.

International Space Station Overhead Notifier

Utilized several APIs to find the exact coordinates of the ISS and whether the sun has set at the user's location. If it is night and the ISS is approximately overhead, this program will send you an email telling you to look up.

Password Manager

Randomly generates a password using a desired number of characters, numbers, and symbols. The program automatically updates a .csv file using Pandas, where all passwords are stored in the JSON format. Search functionality allows you to qickly see what password a certain account uses.

Personal Website

This website is one of my personal favorites. This is one of my first true front-end design experiences, and I love the freedom that comes with being able to manifest a creative vision. HTML, CSS, the Bootstrap framework, Javascript, and jQuery are all tools I used to bring my vision to life.
